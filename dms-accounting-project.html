DMS-Accounting System

A complete accounting document management platform for DR-SI and OR-CR workflows, client tracking, approvals, posting, reporting, and audit-ready operations

Project Overview

DMS-Accounting is a PHP + MySQL financial document management system built to handle sales invoices (DR-SI), payment receipts (OR-CR), client master records, and role-based accounting workflows. It supports complete financial traceability with revision tracking, reporting, scheduled exports, backups, and a compliance-ready audit trail.

Core Features

  • DR-SI (Sales Invoice) encoding, approval, posting, and payment tracking
  • OR-CR (Payment Receipt) recording and invoice allocation
  • Client master file with balance tracking and transaction history
  • Multi-attachment file support with automatic version control
  • Statement of Account (SOA), AR Aging, and summary reports
  • Scheduled report generation with CSV export support
  • User management, role-based permissions, and approval routing
  • Database backup and recovery with integrity verification
  • Full audit trail and admin notifications for key events
  • Responsive pages and PWA support for app-like access

End-to-End Accounting Workflow

  1. Create DR-SI as draft and attach supporting files
  2. Submit for approval and route to authorized approver
  3. Approve and post invoice to accounting records
  4. Create OR-CR when payment is received
  5. Allocate receipt amount to one or more invoices
  6. Post payment and update receivable balances
  7. Generate SOA, AR Aging, and summary reports
